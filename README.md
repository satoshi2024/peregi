合并的思路是：**一个 Case 专门测试 IF 条件成立（差值为 0）走清零逻辑，另一个 Case 专门测试 ELSE 条件（差值不为 0）走原本的带尾数计算逻辑。** 同时在同一个 Case 里把市和县的字段一起验证掉。
> **💡 测试数据的小建议：**
> 在 Case 1 中，强烈建议不要只用 0 - 0 = 0 的数据来测，最好用类似于 均等割额 = 5000, 减免额 = 5000 的数据。这样才能真正证明你写的**减法逻辑**起作用了，而不是仅仅因为默认值就是 0。
