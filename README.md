### 重点日语关键词说明（供参考）：
 * **再現せず (Saigen-sezu):** 无法重现。
 * **事象 (Jishō):** 现象（指 Bug 表现出的现象）。
 * **正常に反映されている (Jōjō ni han'ei sarete iru):** 正常反映（已保存到数据库）。
 * **エビデンス (Ebidentsu):** 证据/截图。
 * **リビジョン (Ribijon):** 版本号（通常对应图片 1 中的“発生リビジョン”）。
 * **切り分け (Kiriwake):** 故障排查/排除变量。
